Studdy_Buddy - Grammar & Sentiment Checker Web App

##Project Objective:
-Studdy_Buddy is a Flask-based web application designed to help students enhance their English writing skills. It checks grammar, identifies sentiment (positive, negative, or neutral), highlights negative sentence segments, and suggests reframed alternatives.


#Tools & Technologies Used:

1. Python
2. Flask
3. TextBlob (for sentiment analysis)
4. LanguageTool (for grammar and spell checking)
5. HTML/CSS (for frontend interface)
6. Bootstrap (for styling)
7. dotenv (.env for API key handling)


#Features

1. Grammar and spell checking
2. Sentiment analysis with polarity and subjectivity scores
3. Identification of negative phrases/sentences
4. Suggestions for positive reframing
5. Stylish and responsive frontend


#How to Run:

1. Clone the repository.
2. Install dependencies from requirements.txt.
3. Create a .env file and add your API keys if needed.
4. Run app.py using Python.
5. Open http://localhost:5000 in your browser.
